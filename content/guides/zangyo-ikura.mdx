残業代は、**1時間あたりの基礎賃金 × 割増率 × 残業時間**で決まります。たとえば月給30万円・所定160時間なら時給は1,875円。時間外労働を月20時間すると、25%増で**約46,875円**の残業代になります。下の早見表で、あなたに近い条件を確認しましょう。

## 1時間あたりの基礎賃金（月の基礎賃金÷所定160時間）

| 月の基礎賃金 | 1時間あたり |
| --- | --- |
| 25万円 | 1,563円 |
| 30万円 | 1,875円 |
| 40万円 | 2,500円 |

## 時間外労働の残業代の早見表（月給30万円・時給1,875円・25%増）

| 時間外（月） | 残業代 |
| --- | --- |
| 10時間 | 約 23,438円 |
| 20時間 | 約 46,875円 |
| 30時間 | 約 70,313円 |
| 45時間 | 約 105,469円 |
| 60時間 | 約 140,625円 |

> 前提：所定労働時間を月160時間とした概算。1か月60時間を超える時間外は50%増になります。あなたの条件は[残業代 計算機](/zangyodai)で試算できます。

## 深夜・休日はさらに上乗せ

- **深夜（22〜5時）**：25%を加算。時間外と重なると合計50%増。
- **法定休日**：35%増。月給30万円なら休日8時間で**約20,250円**。
- **月60時間超の時間外**：50%増。70時間なら合計**約168,750円**。

割増率の詳しい組み合わせは[残業代の計算方法](/guide/zangyo-keisan)で解説しています。

## よくある質問

### 残業代は月いくらになる？
月給30万円で時間外20時間なら約4.7万円、45時間なら約10.5万円が目安です。深夜・休日があればさらに増えます。

### 固定残業代（みなし残業）がある場合は？
固定残業時間を超えた分は別途請求できます（[みなし残業とは](/guide/zangyo-minashi)）。

> 本記事は概算です。基礎賃金に含める手当の範囲などで実際の金額は変わります。正確な計算は勤務先や労働基準監督署にご確認ください。
