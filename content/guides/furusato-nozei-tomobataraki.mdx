共働き夫婦は**配偶者控除がない**ぶん、専業主婦・主夫世帯より一人あたりの上限額が高めです。そして重要なのは、ふるさと納税は**夫婦合算ではなく一人ずつ**で計算し、**それぞれが自分の名義で寄付する**点。二人で行えば、世帯としての返礼品もぐっと増やせます。

## 共働きの上限額（一人あたり・年収別）

共働きで配偶者控除を受けない場合、一人あたりの上限額は「独身」と同じ目安です。

| 本人の年収 | 上限額の目安（一人あたり） |
| --- | --- |
| 300万円 | 約 28,000円 |
| 400万円 | 約 42,000円 |
| 500万円 | 約 61,000円 |
| 600万円 | 約 77,000円 |
| 700万円 | 約 108,000円 |
| 800万円 | 約 131,000円 |
| 1,000万円 | 約 183,000円 |

> 前提：30歳・協会けんぽ（東京）・他の控除なしで、当サイトの[シミュレータ](/furusato-nozei)で試算した目安です。

たとえば**夫600万円・妻500万円**の共働きなら、夫は約7.7万円、妻は約6.1万円、**世帯合計で約13.8万円**まで実質2,000円（各自2,000円・合計4,000円）で寄付できます。各自の正確な額は[ふるさと納税 上限額シミュレータ](/furusato-nozei)で確認しましょう。

## 子どもがいる共働きはどちらかが扶養

高校生（16〜18歳）の子を扶養している場合、**扶養控除はどちらか一方**が受けます。その人は上限額が少し下がる（[年収別早見表](/guide/furusato-nozei-nenshu-betsu)の「夫婦」列が目安）一方、もう一方は「独身・共働き」列のままです。**収入が高いほうが扶養に入れる**と世帯の税負担を抑えやすくなります。なお**中学生以下の子は扶養控除がない**ため、上限額には影響しません。

## 共働きで失敗しないための3つの注意点

1. **名義をそろえる**：寄付の申込名義・支払うクレジットカードは、**控除を受ける本人のもの**にします。妻の控除にしたいのに夫のカードで払うと、夫の寄付として扱われ控除されません。
2. **一人ずつ上限内に収める**：世帯合算で考えないこと。夫の上限・妻の上限はそれぞれ別です。
3. **ワンストップ特例は各自で申請**：二人とも使う場合、**それぞれが寄付先5自治体以内**で、各自が申請書を提出します。手順は[ふるさと納税のやり方](/guide/furusato-nozei-yarikata)へ。

## よくある質問

### 共働きは夫婦合算の年収で計算しますか？
いいえ。ふるさと納税は**一人ずつ**計算します。夫婦それぞれが自分の年収・名義で上限額を確認し、各自で寄付・申請します。

### 共働きと専業主婦世帯で上限はどう違う？
共働きは配偶者控除がないため、同じ年収でも上限額が高めです。たとえば年収500万円なら、共働きは約61,000円、配偶者を扶養する場合は約50,000円が目安です。

### 産休・育休中の配偶者はどう扱う？
その年の収入が少なく配偶者控除の対象になる場合は、もう一方が「夫婦」の列の目安になります。収入が回復した年は各自「独身・共働き」で計算します。

> 本記事の金額は概算です。正確な上限額は源泉徴収票をもとに各自治体・寄付サイトの公式シミュレーションでご確認ください。
